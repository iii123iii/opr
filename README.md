# opr 
 this is a small programming language made with python

## ```print {text or var name}```
print text

## ```play {mp3 location} {nothing or 1 for loop}```
play mp3

## ```stop```
stop mp3

## ```var {var name can only be alpha(only letters)} = {text}```
create a variable

## ```exec {filename.opm}```
to execute files


## ```cls```
clear your terminal on windows

## ```exit```
to exit
